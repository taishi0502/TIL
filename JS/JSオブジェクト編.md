# 配列を作る
```
{
  // const score1 = 80
  // const score2 = 90
  // const score3 = 40

  const scores = [80, 90, 40];
  console.log(scores)
}
```

- 配列要素にアクセス
```
  const scores = [80, 90, 40];
  console.log(scores[1])
```
```
//値の変更
  const scores = [80, 90, 40];
  scores[2] = 44;
  console.log(scores)
```
```
// 配列の要素数を取得
  const scores = [80, 90, 40];
  console.log(scores.length)
```
